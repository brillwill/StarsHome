2.Angular UI Router

客户端单页面应用的一个路由框架

官网资源

Angular UI Router @Github

为什么使用UI Router

  • AJAX请求没法留下History记录
  • 用户无法直接通过URL进入应用中的制定页面(保存书签、分享链接)
  • AJAX对SEO是个灾难
  • AngularJS自带的router不支持嵌套
