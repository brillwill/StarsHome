Selector是一个表达式, 用来选择要添加CSS的HTML元素。最常见的选择器是id选择器(#someId),  和class选择器(.someClass)。实践中,我们经常会根据HTML元素的关系来 组合使用。 Declaration是CSS生命的内容, 它由若干属性和值组成。
这行CSS的意思是, 选择h1标题元素并把它(们)的颜色设为蓝色, 字体大小设为12像素。

CSS用起来

可以通过三种途径来应用CSS

外部样式表

大多采用这种方式, 将CSS存成.css文件并引入到html中

<head>
<link rel="stylesheet" href="mystyle.css">
</head>

内联样式表

混杂在HTML的元素属性中,  工程中不推荐这种, 但有时候调试比较方便。

<p style = "color:blue, margin-left:20px">This is a paragraph.</p>

内部样式表

将css以style元素的方式添加到html中,用的不是很多

<head>
<style>
h1 {color:blue}
</style>
</head>
层叠次序
当同一个HTML的元素被不止一个样式定义时候, 谁会生效呢? 答案是, 在属性的粒度上层叠覆盖, 后出现的优先级高。 一般有以下原则:
  1. 浏览器缺省设置
  2. 外部样式表
  3. 内部样式表
  4. 内联样式
其中内联样式的优先级最高。

CSS布局

Display和Visibility

Display属性用来提示元素如何被显示,会影响布局。Visibility定义元素是否可见,即使不可见也不影响布局。

  • display:none与visibility:hidden
  • display:block 占用全部的宽度,有换行
  • display:inline 占用必要的宽度,无换行

定位Position

表1-1 四中定位方式
序号 设置 是否在正常流中 left/right/top/down/的作用 备注
1 position:static 是 没有作用 默认值
2 position:fixed 否 相对于浏览器的位置
3 position:relative 是 相对于流中正常值的位置

重叠的情况
z-index属性定义了顺序,越大越叠在上面

浮动float

float:left
float:right
元素的水平方向浮动，意味着元素只能左右移动而不能上下移动。
一个浮动元素会尽量向左或向右移动，直到它的外边缘碰到包含框或另一个浮动框的边框为止。
浮动元素之后的元素将围绕它。
浮动元素之前的元素将不会受到影响。

CSS3动画

动画,就是定义某些值得渐进变化过程。

CSS code: 
@keyframes moving {
  from {
    left: 0px;;
  }

  to {
    left:200px;
  }
}

@-webkit-keyframes moving {
  from {
    left: 0px;;
  }

  to {
    left:200px;
  }
}

.show-brick {
  position: relative;
  width:44px;
  height:44px;
  left:0px;
  top:20px;
  background-color: red;
  -webkit-animation:moving 5s linear 3s infinite alternate;
  animation:moving 5s linear 3s infinite alternate;
}

2.Bootstrap

简洁、直观、强悍的前端开发框架，让web开发更迅速、简单。

使用起步

<!-- 新 Bootstrap 核心 CSS 文件 -->
<link rel="stylesheet" href="//cdn.bootcss.com/bootstrap/3.3.5/css/bootstrap.min.css">

<!-- 可选的Bootstrap主题文件（一般不用引入） -->
<link rel="stylesheet" href="//cdn.bootcss.com/bootstrap/3.3.5/css/bootstrap-theme.min.css">

<!-- jQuery文件。务必在bootstrap.min.js 之前引入 -->
<script src="//cdn.bootcss.com/jquery/1.11.3/jquery.min.js"></script>

<!-- 最新的 Bootstrap 核心 JavaScript 文件 -->
<script src="//cdn.bootcss.com/bootstrap/3.3.5/js/bootstrap.min.js"></script>
也可以通过bower install bootstrap或者npm install bootstrap安装到本地
